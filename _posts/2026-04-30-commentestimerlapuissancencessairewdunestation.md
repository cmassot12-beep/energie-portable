Comment évaluer la puissance nécessaire (W) d'une station ?

Une station d'émission électrique (ELE) est un appareil électrique qui permet de générer et de distribuer des émissions de chaleur. Elle peut être composée de deux parties principales : l'électronique de base (BE) et le système de distribution (SD).

Pour évaluer la puissance nécessaire (W) d'une station, il faut connaître les paramètres de l'électronique de base et du système de distribution.

Électronique de base (BE)

L'électronique de base (BE) est constituée de deux composants :

1. Le transistor (T) : Il est le principe de fonctionnement de la station. Le T joue un rôle central dans la distribution des émissions de chaleur. Il est composé de deux segments de résonance (périphérique et périphérique réfractif).

2. Le commutateur (CC) : Il permet de réguler le courant de sortie. Il a une fonction de commutateur. Le CC transmet les courants de sortie dans une direction contrôlable.

Les T sont composés d'un transistormère (TN) et d'un commutateur (CC). Le TN est composé d'un transistor (T) et d'un commutateur (CC). Il peut se faire de deux types : un transistor à PNP (de type P-N) ou un transistor à NPN (de type N-P).

Le commutateur est composé de deux fils (FC) qui transmet les courants de sortie dans une direction contrôlable. L'un des fils peut être en phase (FC1) et l'autre en contre-phase (FC2).

Le commutateur (CC) est composé de deux fils (CC1 et CC2) qui transmet les courants de sortie dans une direction contrôlable.

Le commutateur (CC) peut être configuré en plusieurs modes. Les modes utilisés sont :

- Mode «normale» : Le CC est en mode normal, il transmet les courants de sortie dans une direction contrôlable.
- Mode «périphérique réfractif» : Le CC est en mode périphérique réfractif, il transmet les courants de sortie dans une direction contrôlable dans un sens.
- Mode «périphérique» : Le CC est en mode périphérique, il transmet les courants de sortie dans une direction contrôlable.

Détecter la puissance nécessaire (W)

La puissance nécessaire (W) d'une station peut être déterminée à l'aide du calcul suivant :

- T = 1 W (nombre de transistors)
- CC = 1 W (nombre de commutateurs)
- Périphérique réfractif = CC1 + CC2
- Nombre de transistors = T * CC
- Nombre de commutateurs = Périphérique réfractif * CC

La puissance nécessaire (W) d'une station peut également être calculée à l'aide de la fonction suivante :

P = 2 * CC * T * 1 W (nombre de transistors)

Par ailleurs, il est utile de connaître les performances (W) d'une station pour que les calculs de puissance soient corrects.

Exemple :

Considérons une station de 1 MW dont le commutateur (CC) est composé de 100 fils (CC1 et CC2). La puissance nécessaire (W) est :

P = 100 * CC1 + 100 * CC2 * 1 W (nombre de transistors)
